# Teste OxigenWeb

- needs npm & node installed to run

clone the project:
    
    git clone https://github.com/D0urada/oxigenweb-teste.git

open the folder:

	cd oxigenweb-teste
    
install dependencies:

    npm install

runs the server:
    
	npm start

if you want to generate the dist files, run:

	npm run build 

or

	npm run dev
