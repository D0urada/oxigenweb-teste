# natoo - Teste OxigenWeb

- needs npm & node installed to run

clone the project:
    
    git clone https://github.com/D0urada/natoo.git

open the folder:

	cd natoo
    
install dependencies:

    npm install

runs the server:
    
	npm start

if you want to generate the docs files, run:

	npm run build 

or

	npm run dev
